Bem-Vindo ao Sistema de Lojas

Esse sistema web baseado em Django foi desenvolvido como parte de uma atividade avaliativa para o 3º Bimestre da disciplina de Desenvolvimento de Aplicações Web. A navegação por este pode ser realizado pelo header acima.

O dominio deste sistema é de uma franquia de lojas genérico e as 6 entidades que o constituem são:

  • Cliente
  • Categoria de Produto
  • Produto
  • Pedido
  • Departamento
  • Funcionário
